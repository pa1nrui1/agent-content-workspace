# 选题前检查

## 目标

选题前检查用于避免三个问题：

1. 选题偏离创作者定位。
2. 重复已经做过的母题。
3. 在风险未识别时直接进入写作。

## 检查顺序

推荐任何选题前，Agent 应按以下顺序检查：

1. 创作者定位匹配。
2. 内容主线匹配。
3. 平台适配。
4. 历史查重。
5. 差异化判断。
6. 风险检查。
7. 是否建议继续。

## 历史查重范围

查重至少覆盖：

- 当前平台草稿。
- 当前平台最终稿。
- 当前平台复盘。
- 当前平台样本。
- 当前平台脚本。
- 当前平台预测。
- 其他平台相关内容。
- 全局样本、脚本、预测和复盘。

## 查重问题

Agent 应判断：

- 是否已经做过同一主题。
- 是否只是换了标题。
- 是否属于同一母题。
- 是否仍在讲同一个项目、同一个痛点或同一个结论。
- 是否与近期内容过近。
- 如果继续做，新的差异在哪里。

## 结论

选题前检查只能输出三种结论：

- 建议继续。
- 修改后继续。
- 不建议继续。

Agent 应说明理由和下一步建议。
