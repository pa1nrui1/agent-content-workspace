# 长文开篇检查

## 目标

长文内容需要在标题和开篇建立阅读理由。

长文开篇不应过度铺背景，而应尽快进入问题、争议、判断或真实场景。

## 标题检查

检查项：

1. 标题是否提出明确问题。
2. 标题是否有具体对象。
3. 标题是否让用户知道为什么值得读。
4. 标题是否与正文一致。
5. 标题是否避免夸大和空泛。

## 开篇检查

开篇应完成：

1. 点出问题。
2. 说明争议或判断。
3. 建立与读者的关系。
4. 说明文章将如何展开。

## 结构检查

长文结构应有推进：

1. 问题是什么。
2. 为什么重要。
3. 具体事实或场景。
4. 分析路径。
5. 判断和边界。
6. 实务启示或方法总结。

## 结尾检查

结尾应回到开篇问题，形成判断或行动建议。

结尾不应只是空泛总结。
