# 内容钩子总论

## 定义

内容钩子是内容让用户点击、停留、阅读或观看的关键设计。

不同内容形态有不同钩子：

- 图文内容依赖标题、首图、前两行正文和多图递进。
- 视频内容依赖前三秒、前十秒、节奏推进和结尾讨论口。
- 长文内容依赖标题、开篇问题、结构推进和结尾判断。

## 工作原则

Agent 生成内容后，必须单独检查内容钩子。

内容质量好不等于钩子有效。钩子检查应独立于正文润色。

## 图文内容钩子

图文内容发布前必须检查：

1. 标题是否有点击理由。
2. 首图是否能让用户停留。
3. 前两行正文是否承接标题和首图。
4. 多图之间是否有信息递进。
5. 是否存在隐私或版权风险。

## 视频内容钩子

视频内容发布前必须检查：

1. 前三秒是否有冲突、反差、后果或真实卡点。
2. 前十秒是否说明继续观看的理由。
3. 每 15-20 秒是否有推进。
4. 画面是否辅助口播理解。
5. 结尾是否留下讨论口、下一条线索或边界提醒。

## 长文内容钩子

长文内容发布前必须检查：

1. 标题是否提出值得读的问题。
2. 开篇是否直接进入争议、判断或真实场景。
3. 结构是否能支撑阅读。
4. 结尾是否回到开篇问题。
