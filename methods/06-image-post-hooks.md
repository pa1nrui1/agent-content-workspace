# 图文内容钩子检查

## 目标

图文内容发布前，必须检查标题、首图、前两行正文和多图结构。

标题负责提供点击理由。

首图负责让用户停留。

前两行正文负责承接标题和首图。

多图结构负责形成信息递进。

## 标题检查

检查项：

1. 标题是否有具体对象。
2. 标题是否有真实痛点。
3. 标题是否有明确反差。
4. 标题是否能让用户知道为什么要点开。
5. 标题是否与正文内容一致。
6. 标题是否存在过度承诺。

## 首图检查

检查项：

1. 首图是否一眼能看懂主题。
2. 首图是否有清楚视觉中心。
3. 首图是否与标题互相补充。
4. 首图文字是否过多。
5. 首图是否具有真实感、信息价值或审美吸引力。
6. 首图是否包含隐私信息或版权风险。

## 前两行正文检查

检查项：

1. 是否重复标题。
2. 是否进入具体场景。
3. 是否说明这个问题为什么值得看。
4. 是否像说明书。
5. 是否有明确对象。

## 多图结构检查

多图内容不应硬凑张数。

常见结构：

1. 首图：给点击理由。
2. 第二图：说明问题。
3. 第三图：展示过程。
4. 第四图：展示结果。
5. 第五图：说明边界。
6. 第六图：总结或行动建议。

检查项：

- 每张图是否有独立信息。
- 图片之间是否递进。
- 是否存在重复图。
- 是否存在无信息量的装饰图。
- 是否需要用户确认视觉方向。
