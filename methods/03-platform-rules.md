# 平台规则配置

## 目标

不同平台需要不同表达方式。

平台规则用于规定每个平台适合什么内容、不适合什么内容、如何开头、如何组织结构、如何检查内容钩子和发布边界。

## 抽象平台

本模板默认使用三个抽象平台：

- 平台 A：长文平台。
- 平台 B：图文平台。
- 平台 C：短视频平台。

用户可以根据实际情况重命名或新增平台。

## 每个平台应配置的内容

每个平台至少应配置：

1. 平台定位。
2. 目标受众。
3. 适合内容类型。
4. 不适合内容类型。
5. 常用结构。
6. 标题规则。
7. 开头规则。
8. 视觉规则。
9. 发布前检查项。
10. 风险边界。

## Agent 行动

Agent 在写作前必须确认目标平台。

确认平台后，Agent 应读取对应平台规则，并按该平台规则生成或检查内容。

如果用户要求跨平台改写，Agent 应重新判断目标平台表达方式，不能直接复制原平台内容。
