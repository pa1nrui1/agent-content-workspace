# 禁止公开信息清单

以下信息默认不得进入公开仓库。

## 个人信息

- 真实姓名
- 手机号
- 邮箱
- 地址
- 身份证号
- 执业证号
- 头像
- 真实账号名

## 工作信息

- 客户名称
- 当事人名称
- 案号
- 合同
- 聊天记录
- 内部沟通
- 未公开项目
- 平台后台数据

## 技术信息

- token
- cookie
- secret
- API key
- auth 文件
- 登录缓存
- 本地绝对路径
- 未脱敏配置文件

## 内容资产

- 未发布草稿
- 真实最终稿
- 真实复盘数据
- 真实预测日志
- 真实平台样本
- 第三方文章原文
- 未授权文风样本
