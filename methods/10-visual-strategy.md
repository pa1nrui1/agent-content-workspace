# 视觉策略判断

## 目标

视觉策略用于决定一条内容应使用什么视觉形式。

Agent 不得默认真实截图、信息卡、生成图或 HTML 幻灯片一定适合所有内容。

## 判断维度

制作视觉素材前，应判断：

1. 内容类型。
2. 目标平台。
3. 用户参考案例。
4. 内容是否需要证据展示。
5. 内容是否需要流程展示。
6. 内容是否需要情绪冲击。
7. 内容是否需要收藏价值。
8. 是否存在隐私风险。
9. 是否需要第三方 Skill。

## 常见视觉形式

可选形式包括：

- 真实截图。
- 信息卡。
- 流程图。
- 对比图。
- 封面海报。
- HTML 幻灯片。
- 生成图。
- 真实照片。
- 文字卡片。

## 使用真实截图

真实截图适合：

- 展示真实工具。
- 展示真实过程。
- 展示文档结构。
- 展示操作结果。

使用前必须脱敏。

## 使用信息卡

信息卡适合：

- 清单。
- 步骤。
- 知识点。
- 对比。
- 概念解释。

## 使用 HTML 幻灯片

HTML 幻灯片适合：

- 视频辅助画面。
- 流程演示。
- 项目结构展示。
- 复杂概念可视化。

## 使用生成图

生成图适合：

- 辅助视觉。
- 概念背景。
- 封面氛围。

生成图不得冒充真实截图或真实运行结果。
