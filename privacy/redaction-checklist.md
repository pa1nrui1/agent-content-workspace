# 脱敏检查表

公开任何文件、截图、示例、模板或演示前，必须检查以下项目。

## 一、身份信息

- [ ] 真实姓名
- [ ] 真实账号
- [ ] 头像
- [ ] 电话
- [ ] 邮箱
- [ ] 地址
- [ ] 身份证号
- [ ] 执业证号

## 二、业务和客户信息

- [ ] 客户名称
- [ ] 当事人信息
- [ ] 案号
- [ ] 合同
- [ ] 聊天记录
- [ ] 平台后台数据
- [ ] 真实交易数据
- [ ] 未公开项目数据

## 三、技术敏感信息

- [ ] token
- [ ] cookie
- [ ] secret
- [ ] auth 信息
- [ ] API key
- [ ] 本地绝对路径
- [ ] 账号登录状态

## 四、版权风险

- [ ] 第三方文章原文
- [ ] 未授权图片
- [ ] 未授权视频
- [ ] 未授权文风样本全文
- [ ] 未保留许可证的第三方代码或 Skill

## 五、处理方式

- 使用占位符替换真实信息。
- 截图中遮挡敏感信息。
- 第三方内容只保留链接、摘要、风格卡或许可证允许的内容。
- 无法判断是否可公开时，先询问用户。
