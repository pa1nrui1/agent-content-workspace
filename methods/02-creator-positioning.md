# 创作者定位配置

## 目标

创作者定位用于回答三个问题：

1. 创作者是谁。
2. 创作者长期说什么。
3. 创作者不说什么。

定位配置完成后，Agent 才能判断一个选题是否适合当前工作区。

## 配置内容

创作者定位至少包括：

- 对外身份。
- 专业背景。
- 目标受众。
- 内容领域。
- 长期表达方向。
- 表达边界。
- 禁止公开的信息。
- 需要人工确认的内容。

## 判断标准

一个好的创作者定位应满足：

1. 有明确对象，不是面向所有人。
2. 有长期问题意识，不是追逐临时热点。
3. 有表达边界，不把所有事情都纳入内容范围。
4. 有可信来源，能说明创作者为什么能谈这个话题。
5. 有可持续内容主线，能持续拆分选题。

## Agent 行动

当用户提出新选题时，Agent 应先判断：

- 选题是否符合创作者定位。
- 是否属于已有内容主线。
- 是否超出创作者表达边界。
- 是否需要补充新的定位配置。

如果选题偏离定位，Agent 应说明偏离点，并给出调整建议。
