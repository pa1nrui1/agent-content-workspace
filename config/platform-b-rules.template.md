# 平台 B 规则

## 一、平台定位

平台 B 用于承载图文、图片卡片、过程展示、工具展示和收藏型内容。

- 平台类型：图文平台
- 核心受众：
- 内容目标：
- 适合内容：
- 不适合内容：

## 二、内容类型

平台 B 主要支持：

1. 图文笔记。
2. 多图卡片。
3. 过程展示。
4. 清单和方法总结。
5. 视觉化案例。

## 三、图文内容结构

图文内容应重点检查：

1. 标题。
2. 首图。
3. 前两行正文。
4. 多图递进。
5. 收藏价值。

## 四、标题规则

标题应具备：

- 具体对象。
- 真实痛点。
- 明确反差。
- 点击理由。

标题不得：

- 夸大效果。
- 制造虚假焦虑。
- 暗示不可实现的结果。
- 与图片和正文不一致。

## 五、首图规则

首图负责让用户停留。

首图应检查：

- 是否一眼能看懂主题。
- 是否有清楚视觉中心。
- 是否与标题互相补充。
- 是否文字过多。
- 是否包含隐私或版权风险。

## 六、视觉规则

视觉形式不固定。可以使用：

- 真实截图。
- 信息卡。
- 流程图。
- 对比图。
- 封面海报。
- 生成图。

使用前必须确认视觉方向。
